# Categories Reference — Latino Connect Hub

This document defines the full category taxonomy for the platform. It is structured in two levels:
- **Groups** — broad areas shown on the directory home page (with icon + colour)
- **Categories** — the actual selectable value when a business registers

The goal is to cover every type of business a Latin entrepreneur might run in New Zealand.

## Implementation Notes

- The `key` values above are the stable DB slugs (snake_case, English).
- Each key needs a translation entry in `modal.categories.<key>` in all three locale files.
- The **Group** level maps to the visual category grid on the directory home (`DirectoryHome.tsx`).
- The **Category** level is what the business owner selects during registration and what gets stored in the `businesses.category` column.
- If a business doesn't fit any category, `other` remains as a fallback.
- Recommended rollout: start with the groups and the categories that already have demand in your network, then expand.
